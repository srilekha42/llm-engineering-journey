# 🧠 Day 1 Notes – LLM Basics

---

## 🔹 What is an LLM?

An LLM (Large Language Model) is an AI model that generates text based on input prompts.

---

## 🔹 Core Flow

Prompt → AI Model → Response

---

## 🔹 Key Concepts

### 1. Prompt
The instruction we give to the AI.

Example:
"Explain AI in simple words"

---

### 2. System vs User Prompt

- System Prompt → defines behavior (e.g., "You are a helpful assistant")
- User Prompt → actual task or question

---

### 3. API Usage

Steps:
1. Load API key using `.env`
2. Send prompt to model
3. Receive response
4. Print output
